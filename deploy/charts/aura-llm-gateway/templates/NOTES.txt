Aura LLM Gateway has been installed.

1. Check the deployment status:

   kubectl get pods -n {{ .Release.Namespace }} -l "app.kubernetes.io/name={{ include "aura-llm-gateway.name" . }},app.kubernetes.io/instance={{ .Release.Name }}"

2. Forward a port to test locally:

   kubectl port-forward -n {{ .Release.Namespace }} svc/{{ include "aura-llm-gateway.fullname" . }} 8080:{{ .Values.service.port }}

3. Hit the health endpoint:

   curl http://localhost:8080/health

    openssl rand -hex 32
